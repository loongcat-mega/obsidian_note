#### 捕获列表[]
捕获非静态局部变量
捕获值，捕获的变量不能修改，加mutable可以修改,但再次访问会继承上次修改的值，若将Lambda表达式看成仿函数，则捕获值相当于类的成员变量
```c++
int x=0;
auto foo=[x]()mutable{x++;};
foo();//x=1
foo();//x=2
foo();//x=3
```
捕获引用可以修改
```c++
auto foo=[&x]{x++;};
```

```c++
[this] 捕获this指针
[=]捕获lambda表达式定义作用域的全部变量的值
[&]捕获lambda表达式定义作用域的全部变量的引用

```
