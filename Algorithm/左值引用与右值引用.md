`等号左边的值即左值，等号右边的值即右值；但为了更加严谨，我们这样定义：能够取地址的，有名字的就是左值；反之，不能取地址，没有名字的就是右值。`

## 左值引用
```c++
int &a=b;
//b修改，a同步修改
```
## 右值引用
```c++
pair<int,int>hashmap[10];
auto&&[fst,snd]=hashmap[1];
//修改fst 与 snd的值，hashamap[1]的值同步修改
```

```c++
i++; 右值 不能取地址
++i; 左值 能取地址
```


常量左值引用不止能引用左值，还能引用右值
```c++
const int&a=10;
const int&b=a;

```

复制构造器：左值引用
移动构造器：右值引用
